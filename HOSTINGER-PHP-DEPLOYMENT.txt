MITOMS - HOSTINGER SHARED HOSTING DEPLOYMENT
============================================

This project uses:
- Next.js only to create a static frontend build on your computer.
- PHP for the consultation/contact form backend on Hostinger.
- No Node.js process is required on the Hostinger server.

REQUIREMENTS ON YOUR COMPUTER
-----------------------------
1. Node.js 20.9 or newer.
2. npm, included with Node.js.

WINDOWS BUILD
-------------
1. Extract the source ZIP.
2. Double-click build-hostinger.bat.
3. Wait for the build to complete.
4. The script creates MITOMS-PublicHTML-Build.zip.

You can also run these commands in PowerShell:

  npm install
  npm run build

The generated website will be in the out folder.

HOSTINGER UPLOAD
----------------
1. Open Hostinger hPanel.
2. Open Websites > Manage > File Manager.
3. Open public_html.
4. Back up and remove the old website files.
5. Upload MITOMS-PublicHTML-Build.zip.
6. Extract it directly inside public_html.

The final structure must contain:

  public_html/index.html
  public_html/.htaccess
  public_html/_next/
  public_html/api/consultation.php

Do not leave the files inside an additional folder.

EMAIL CONFIGURATION
-------------------
Open public/api/consultation.php before building and check:

  CONSULTATION_TO_EMAIL
  FROM_EMAIL
  FROM_NAME

The default recipient and sender are sales@mitoms.com.
Create this mailbox in Hostinger before testing the form.

IMPORTANT
---------
- Never upload the source-code ZIP directly into public_html.
- Upload the generated MITOMS-PublicHTML-Build.zip instead.
- The PHP endpoint is copied into out/api/consultation.php during npm run build.
- Local Next.js development does not execute PHP. Test email delivery after uploading to Hostinger, or run a separate local PHP server for backend testing.
